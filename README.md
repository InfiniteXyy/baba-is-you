# Baba Is You

> 🧪 **This project is an experiment** — built entirely with [Claude](https://claude.ai) (Anthropic's AI assistant) as a learning exercise to explore AI-assisted game development. It is **for educational and learning purposes only**.

🎮 **[Play it here →](https://InfiniteXyy.github.io/baba-is-you/)**

A web-based recreation of the puzzle game *Baba Is You*, where you push words around to change the rules of the game. Built with React, TypeScript, and Vite.

## Features

- 🧩 18 playable levels from the community level format
- 🎨 Sprite-based rendering with animated tiles from the original spritesheet
- 🔄 Full rule engine: parse "X IS Y" patterns to change game behavior
- ✏️ Built-in map editor with community format import/export
- 📱 Keyboard controls (Arrow keys / WASD, Z to undo, R to restart)
